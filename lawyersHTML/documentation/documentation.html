1. Getting Started

1.1 General Info

In order to start modifing the template go to lawyers-html folder in the download package. The general css file is styles.css file.

1.2 Pages Big Image

In order to change the top big image, open an individual file (for example about.html). Find the line <div class="page-head" style="background-image:url(images/bkgs/about.jpg);"> and change the image path and name (if you want).

1.3 Home Slider

Open index.html and go to line 69. There is the beginning of home slider. Change slides, text for each slider.

If you want to modify slider parameters go to line 3 in js/init.js file.

1.4 Practice Areas Page

Open practice-areas.html . Here you can add or remove items. To add a new practice item, you'll have to create a new row at the end, line 246. Then insert the new item. The code below will help you.

<div class="row">
<div class=" col-sm-6 col-md-3">
<div class="practice-item">
<a data-toggle="modal" href="practice-area-single.html" data-target="#myModal">
<div class="practice-icon"><i class="fa fa-bar-chart-o"></i></div>
<h5 class="practice-title">Corporate Finance</h5>
</a>
</div><!--.practice-item-->
</div><!--.col-md-3-->
</div><!--.row-->

Now to create the more info page for this new practice item, you'll have to duplicate the practice-area-single.html page. Give it a unique name, then insert the page name (see the code above) into the href attribute of the practice item.

1.6 Change Contact & Free Evaluation Form Email

You have to edit the include/contact-process.php include/evaluation-process.php and change the $recipient variable value with your own email address.

2. Files Used

2.1 CSS Files

All CSS files, with the exception of the main CSS file (style.css) can be found in the theme's /css/ folder.

style.css - Main stylesheet and determines the majority of the templates styling.

bootstrap.css & bootstrap-responsive.css - The styles for the Twitter Bootstrap framework.
